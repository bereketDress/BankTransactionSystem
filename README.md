Bank Transaction System (Java)
A console-based banking application developed in Java that supports core banking operations such as deposits, withdrawals, and fund transfers. The system is built using Object-Oriented Programming (OOP) principles to ensure modularity, scalability, and maintainability.
It includes robust input validation and error handling to ensure secure and reliable transaction processing. Comprehensive testing has been performed to verify system accuracy and stability.
 Features
 	Deposit, withdrawal, and transfer functionality
 	Object-oriented design (encapsulation, inheritance, polymorphism)
 	Input validation and exception handling
 	Secure and reliable transaction processing
 	Console-based user interface
